# Klantportaal — PartsProfi

Klantportaal dat aansluit op de PartsProfi CRM. Klanten kunnen hier hun bestellingen en onderdelen volgen, een leverdatum kiezen of verplaatsen, en de status van hun order zien.

Huidige fase: fundament (datastructuur + mockdata), nog geen koppeling met echte systemen. Zie `.claude/plans` in het bronproject voor het volledige bouwplan.

## Starten

```
npm install
npm run dev
```

## Later te koppelen
- Orderdatabase / CRM-backend (status-updates)
- Boekhoudsysteem (facturatie, prijzen)
- DHL-tracking (verzendstatus)
- Betaalprovider (betaalstatus)
